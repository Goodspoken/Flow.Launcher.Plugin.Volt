# Volt ⚡

**Volt** — это легковесный плагин для [Flow Launcher](https://www.flowlauncher.com/), позволяющий мгновенно переключать схемы электропитания в Windows 11.

![Volt Icon](Images/app.png)

## Особенности

- 🚀 **Быстрота:** Переключайте режимы одной командой без захода в Панель управления.
- ⚡ **3 Режима:** Поддержка стандартных схем Windows (Максимальная производительность, Сбалансированный, Энергосбережение).
- 🎨 **Понятный UI:** Цветные индикаторы для каждого режима.

## Использование

1. Вызовите Flow Launcher (обычно `Alt + Space`).
2. Введите ключевое слово `volt`.
3. Выберите нужную схему из списка и нажмите `Enter`.

## Установка

### Через магазин плагинов (скоро)
Введите в Flow Launcher:
`pm install Volt`

### Вручную (Direct Install)
1. Скачайте последний релиз из раздела [Releases](https://github.com/Goodspoken/Flow.Launcher.Plugin.Volt/releases).
2. Распакуйте архив в папку плагинов Flow Launcher:
   `%APPDATA%\FlowLauncher\Plugins\Volt`
3. Перезапустите Flow Launcher.

## Требования

- Windows 10/11
- Python 3.x (устанавливается вместе с Flow Launcher)

## Разработка

Плагин написан на Python с использованием стандартной утилиты `powercfg`.
Зависимости забандлены в папку `lib` для работы "из коробки".
